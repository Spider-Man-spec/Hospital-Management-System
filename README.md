# Hospital Management System 🏥

A console-based Hospital Management System developed in C that helps manage patient records, emergency cases, billing, and discharge summaries. The project demonstrates the use of structures, arrays, stacks, functions, and menu-driven programming concepts.

## Features ✨

* Add Patient Records
* View All Patients
* Manage Emergency Cases Using Stack
* Treat Emergency Patients
* View Emergency Stack
* Generate Patient Bills
* View Billing Records
* Discharge Patients
* Store and View Discharge Summaries

## Technologies Used 🛠️

* C Programming
* Structures
* Arrays
* Stacks
* Functions
* Menu-Driven Programming

## How to Run 🚀

1. Compile the program:

   ```
   gcc hospital.c -o hospital
   ```

2. Run the executable:

   ```
   ./hospital
   ```
